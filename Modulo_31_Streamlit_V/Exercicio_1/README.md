# Modulo 31 | Streamlit V | Exercicio 1

**Por:** Weslei Granza  
**Data:** 28 de maio de 2026

## Descricao

Aplicacao em Streamlit para segmentacao de clientes usando RFV: Recencia, Frequencia e Valor. O app permite carregar uma base CSV/XLSX ou usar o arquivo demonstrativo presente na pasta `input`.

## Links

- Aplicacao no Render: a definir apos o deploy
- Repositorio GitHub: https://github.com/NzATrue/EBAC---Weslei-Granza/tree/main/Modulo_31_Streamlit_V/Exercicio_1

## Como executar localmente

```bash
pip install -r requirements.txt
streamlit run app_RFV.py
```

## Deploy no Render

Conectar este repositorio no Render e usar a pasta `Modulo_31_Streamlit_V/Exercicio_1` como Root Directory.

```bash
streamlit run app_RFV.py --server.port $PORT --server.address 0.0.0.0
```
